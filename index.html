BookMyShow Clone

This project is a frontend clone of the popular movie booking platform, BookMyShow, created using HTML and CSS. It replicates the basic structure and design of the website, providing a simple and visually appealing layout for users to browse movies and events.

Features:

  • Home Page: Displays a hero section with trending movies, an eye-catching layout, and call-to-action buttons.
  • Movie List: A grid-based layout showcasing various movie posters, movie names, and basic details like genres and ratings.
  • Responsive Design: The layout is mobile-friendly, ensuring a smooth user experience on different screen sizes, utilizing media queries and CSS Grid/Flexbox for responsiveness.
  • Navigation Bar: Easy-to-use navigation for seamless browsing between different pages such as Movies, Events, and Contact.
  • Footer Section: Contains copyright information and links to external resources.

Technologies Used:

  • HTML for structure and content
  • CSS for styling and layout
  • Flexbox and Grid for responsive design

Solar Planet

This project showcases a simple and interactive solar system simulation built using HTML and CSS. It features a series of planets orbiting a central sun, with each planet styled and animated to move around the sun in a circular path.

Key Features:

  • Interactive animation of planets orbiting a sun.
  • Each planet has a unique color and size representing different celestial bodies.
  • CSS animations are used to make the planets rotate around the sun.
  • Responsive design to ensure the layout adjusts to different screen sizes.

Technologies Used:

  • HTML: Structure of the solar system and planets.
  • CSS: Styling, layout, and animation of planets and sun.

How It Works:

The planets are represented using div elements with circular shapes achieved through the border-radius property. CSS animations are applied to create the effect of planets orbiting the sun. The planets move in a circular path using rotate() and translate() within the CSS keyframes animation.
